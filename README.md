This project is a tool that tests Pion's SCTP implementation, with the following examples:
- *Any* Pion SCTP version can be tested
- Simulate various networking conditions
  - Dynamic RTT
  - UDP Packet Duplication
  - Packet Loss
  - Un/ordered packets
- And more...
